# **App Name**: SolBlaze

## Core Features:

- Wallet Connection: Connect to Solana wallet (Phantom, Solflare, Backpack support).
- Asset Auto-Fetch: Auto-fetch standard NFTs (including unverified), cNFTs (via Helius), and SPL tokens from the connected wallet.
- Asset Display: Display fetched assets in a clear and categorized dashboard (NFTs, cNFTs, Tokens).
- Asset Selection: Allow the user to select any asset (NFT, cNFT, token) from a dropdown menu, populating the asset's metadata.
- Asset Transfer: Enable transfer of selected NFTs, cNFTs, or SPL tokens to a recipient address.
- Asset Burning: Enable burning of selected SPL tokens, NFTs, or cNFTs after confirmation.
- Transaction Display: Display transaction result and link to Solana Explorer for all transfer/burn actions.

## Style Guidelines:

- Primary color: Vibrant purple (#A020F0) to convey innovation and trust.
- Background color: Light gray (#F0F0F0) to create a clean, modern interface.
- Accent color: Electric blue (#7DF9FF) to highlight interactive elements.
- Body and headline font: 'Inter' sans-serif for a modern, objective feel.
- Use crisp, outlined icons for asset types and actions.
- Use a modular layout with clear separation between sections for asset display, selection, and actions.
- Subtle animations for loading states and transaction confirmations.
